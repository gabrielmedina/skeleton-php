# Skeleton

> Inspired in [Skeleton Project Rails](https://github.com/luizpicolo/skeleton-project-rails)

## How to use

### Required
- Node.js (Gulp and Bower)
- Ruby (Sass)

Dependencies of NPM

```shell
npm install
```

Dependencies of Bower

```shell
bower install
```

Run Gulp

```shell
gulp
```

:ship:
