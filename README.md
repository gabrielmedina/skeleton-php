# Skeleton

> Inspired in [Skeleton Project Rails](https://github.com/luizpicolo/skeleton-project-rails)

## How to use

### Required
- Node.js (Grunt and Bower)
- Ruby (Sass)

Dependencies of NPM

```shell
npm install
```

Dependencies of Bower

```shell
bower install
```

Run Grunt

```shell
grunt w
```

:ship:
